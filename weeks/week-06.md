# 第 6 周：Function Calling / Tool Calling

## 本周目标

让模型不仅能聊天，还能调用后端工具。

## 学习内容

- Function Calling
- Tool Schema
- 工具白名单
- 参数校验
- 权限控制
- 工具调用日志

## 具体任务

### Day 1

- [ ] 学习 Function Calling 流程
- [ ] 定义第一个工具：查询天气或查询订单
- [ ] 写工具 JSON Schema

### Day 2

- [ ] 后端实现工具注册机制
- [ ] 模型返回工具调用时执行对应函数
- [ ] 把工具结果返回给模型

### Day 3

- [ ] 实现 3 个业务工具
- [ ] 示例：查客户、查订单、生成邮件
- [ ] 增加参数校验

### Day 4

- [ ] 前端展示工具调用过程
- [ ] 展示工具入参和结果
- [ ] 记录工具调用日志

### Day 5

- [ ] 增加工具白名单
- [ ] 增加错误处理
- [ ] 增加超时控制

### 周末

- [ ] 整理 Function Calling 流程图
- [ ] 写面试问答笔记

## 产出物

- 一个可调用工具的 AI 助手

## 验收标准

- [ ] 模型能根据用户意图选择工具
- [ ] 工具执行结果能进入最终回答
- [ ] 前端能看到工具调用记录

## 面试准备

你要能讲清楚：

- Function Calling 完整流程
- 工具调用为什么需要白名单
- 如何处理工具执行失败
