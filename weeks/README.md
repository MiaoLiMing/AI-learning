# 12 周学习路线

每周都按“学习目标、具体任务、产出物、验收标准”执行。

## 总览

| 周次 | 主题 | 核心产出 |
| --- | --- | --- |
| 第 1 周 | Python + FastAPI + LLM API | 最小聊天 API |
| 第 2 周 | Vue AI 聊天界面 + 流式输出 | 可用聊天应用 |
| 第 3 周 | 文档上传与解析 | 文档入库服务 |
| 第 4 周 | Embedding + 向量检索 | 基础 RAG 检索 |
| 第 5 周 | RAG 问答系统完善 | 企业知识库 MVP |
| 第 6 周 | Function Calling | 工具调用 Demo |
| 第 7 周 | LangGraph 工作流 | 多步骤 Agent |
| 第 8 周 | Agent 平台完善 | 业务 Agent MVP |
| 第 9 周 | 工程化：权限、日志、成本 | 可维护 AI 服务 |
| 第 10 周 | Docker 部署与质量优化 | 一键部署项目 |
| 第 11 周 | 简历与作品集 | 求职材料 |
| 第 12 周 | 模拟面试与投递 | 开始投递 |
