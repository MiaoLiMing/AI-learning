技术栈选择：

```
前端 Vue3

后端 FastAPI

数据库 PostgreSQL

向量库 pgvector
```
目标：
```
支持PDF上传、知识库管理、问答、引用来源。
```

流程：
```
上传 PDF
  ↓
解析文本
  ↓
清洗与切片
  ↓
生成 Embedding
  ↓
写入向量数据库
  ↓
用户提问
  ↓
问题 Embedding
  ↓
向量检索
  ↓
权限过滤 / 重排序
  ↓
拼接上下文
  ↓
LLM 回答
  ↓
引用来源
```

上传流程：
```
上传 PDF
  ↓
保存到对象存储
  ↓
数据库创建 document，status = pending
  ↓
投递异步任务
  ↓
立即返回 document_id
```
