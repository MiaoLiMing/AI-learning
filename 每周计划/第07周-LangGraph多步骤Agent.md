# 第 7 周：LangGraph 多步骤 Agent

## 本周目标

用 LangGraph 把工具调用升级为可追踪、可控制的多步骤 Agent 工作流。

## 能力目标

- 能解释 State、Node、Edge、Graph。
- 能实现“模型节点 -> 工具节点 -> 模型节点”的循环。
- 能保存 Agent 执行轨迹。
- 能说明为什么复杂 Agent 需要工作流。
