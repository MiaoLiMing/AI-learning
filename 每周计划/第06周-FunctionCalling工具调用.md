# 第 6 周：Function Calling / 工具调用

## 本周目标

让模型不仅能回答，还能根据用户意图调用后端工具。

## 能力目标

- 能解释 Function Calling 的完整流程。
- 能定义 Tool Schema。
- 能做参数校验、工具白名单、错误处理。
- 能在前端展示工具调用过程。
