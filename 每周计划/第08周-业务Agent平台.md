# 第 8 周：业务 Agent 平台

## 本周目标

把 Agent Demo 做成可展示的业务平台，重点补人工确认、任务持久化、失败重试和执行历史。

## 能力目标

- 能设计 Agent 任务表和步骤表。
- 能实现人工确认节点。
- 能处理失败重试和执行恢复。
- 能把 Agent 包装成企业用户能理解的界面。

| Day 1 | 任务数据结构 | 任务表、步骤表 | 状态、持久化 |
